# gardusig/static-puzzles-docs

Product and game specifications for static-puzzles — markdown sources plus plain-text mirrors.

## Quick start

```bash
make test
```

## Layout

- `docs/GAMES.md` — issue todo table and workflow
- `docs/REQUIREMENTS.md` — per-game Must/Should/Data checklists
- `docs/GOALS.md` — product scope and success criteria
- `docs/*.txt` — plain-text mirrors with the same facts
